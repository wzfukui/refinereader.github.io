Refine Reader Privacy Policy

Last Updated: 2025-02-03

Data Collection and Usage

Refine Reader takes your privacy seriously. We only collect the following necessary information:

  1. Page Content: Only when you actively click the "AI Summary" button, we get the current tab's article content for summary generation.
  2. User Settings:
    • AI service configuration (API keys, etc.)
    • Theme preferences
    • Prompt settings
    This information is stored locally in your browser and is never uploaded to any servers.

Data Storage

  • All configuration information uses Chrome browser's sync storage
  • All data is saved locally only
  • We do not collect or upload any personal information

Third-party Services

This extension supports multiple AI service providers (such as OpenAI, Tongyi Qianwen, etc.). When using these services:

  • You need to comply with their respective privacy policies
  • Only necessary article content is transmitted for summary generation
  • No personal information is transmitted

Data Security

  • Sensitive information like API keys uses browser's secure storage
  • All network requests use HTTPS encryption
  • No data collection or transmission in the background
